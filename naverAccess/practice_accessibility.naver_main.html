간편하고 정확한 검색

  • 다양한 검색
  • 검색 제안
  • 키패드 슬라이더

검색 버튼 하나로 다양한 기능을 펼쳐볼 수 있어요!

검색창 우측의 마이크 아이콘(인식검색버튼)을 누르면 음성으로 검색하기, 음악 검색,
바코드 검색, 와인 검색, 이미지로 외국어 검색까지 간편하게 검색할 수 있어요.

오탈자도 자동으로 수정된 문자를 제안해줘요!

작은 키패드로 문자를 입력하거나, 음성으로 검색을 하다 보면 오타가 생기는 경우가 많죠?
잘 모르는 단어나 오타는 검색 제안을 통해 정확한 검색을 할 수 있어요.

작은 입력 창의 초점도 쉽게 맞출 수 있어요!

모바일에서 URL 입력 도중 수정해야 하는 텍스트가 생기면, 키패드 슬라이더 기능을 이용해서 쉽게 초점을 이동하세요.
네이버 앱 메뉴창을 열어 ‘네앱연구소’에 들어가시면 더 많은 기능을 설정할 수 있어요.
