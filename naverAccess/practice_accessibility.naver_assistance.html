스크린리더 환경 설정

Sense Reader(센스리더)

1. 가상커서 환경설정(ctrl + shift + F9)을 호출하여 숨긴 내용 읽기를 해제합니다.
2. 가상커서 설정에서 새 페이지 자동 읽기를 ‘읽지 않기’ 또는 ‘간략하게 읽기’로 설정합니다.
3. 가상커서 설정에서 셀 주소 읽기를 뒤에 읽기로 설정합니다. 이렇게 하면 테이블 탐색시에 병합된 셀 파악이 가능합니다.
4. Internet Explorer 메뉴 > 도구 > 인터넷 옵션 > 보안 탭에서 ‘인터넷 보호모드’ 사용을 해제하면 센스리더에서 제공하는 가상커서의 임시 해제(ctrl + 아래쪽 화살표)를 사용할 수 있습니다.

MacOS

1. VoiceOver 및 VoiceOver 유틸리티를 사용하려면 키보드 환경설정에서 '모든 F1, F2 등의 키를 표준 기능 키로 사용'을 켜야 합니다. 다른 일부 Mac 키보드 단축키를 사용하려면 VoiceOver가 다음 키 누르기를 무시하도록 설정해야 할 수 있습니다.
2. command + F5 또는 fn + command + F5 단축키로 ViceOver를 켜거나 끕니다.
3. VoiceOver가 켜져 있는 경우 VoiceOver 유틸리티를 여는 단축키는 control + option + F8 또는 fn + control + option + F8입니다.

네이버 메일

네이버 메일 > 환경설정에서 단축키 사용을 체크하면 읽기 쓰기 등에서 여러 단축키를 사용할 수 있으며 사용자가 원하는 단축키로 변경도 가능합니다.
관련 단축키 리스트는 메일 > 환경설정 > 단축키 설정을 참고하시기 바라며, 스크린리더 사용시 네이버 단축키와 스크린리더 기능키가 중복될 수 있으므로 가상커서를
임시 해제하거나 스크린리더 단축키를 일시 중지해야 합니다.

동영상 서비스

동영상에 자막 올리기

네이버에서는 카페나 블로그 등에 동영상을 게시할 때 smi, srt 확장자를 가진 자막파일을 업로드하여 동영상에 자막을 제공할 수 있습니다.
이러한 자막 서비스는 외국인뿐만 아니라 시각·청각장애인에게도 도움이 됩니다.

자막 올리기

1. 블로그, 카페 등의 글쓰기에서 동영상을 업로드한 후 해당 동영상을 재생합니다.
2. 자막 업로드 버튼이 생성됩니다.
3. 업로드할 자막을 추가하고 영상을 새로고침하면 바로 자막이 적용된 것을 확인할 수 있습니다.

현재는 크롬 브라우저에서만 자막 텍스트에 접근이 가능하며 자막이 갱신될 때마다 제공되는 자막읽기 기능은 지원되지 않습니다.

네이버 동영상 사용안내

• 동영상 플레이 첫 페이지에서 esc키로 플레이어를 음소거 할 수 있습니다.
• 동영상 플레이어에 포커스 후 스페이스바 키를 누르면 일시 정지/재생할 수 있습니다.
• 동영상 플레이어에 포커스 후 좌/우 화살표 키를 누르면 이전으로/다음으로 재생할 수 있습니다.
